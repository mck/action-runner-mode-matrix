# Mode Matrix

Figma Action Runner action that creates a visual permutation matrix of variable modes.

Select frames in Figma, run the action — it clones your selection for every mode in a variable collection, labels each column, and groups the result.

## Usage

Add this repo as a tap in Action Runner:

```
owner: mck
repo: action-runner-mode-matrix
```

Or point the dev server at this directory:

```bash
npx @action-runner/dev ./
```
